## 📦 Installation

### Clone Repository

```bash
git clone https://github.com/your-username/portfolio.git
```

### Install Frontend Dependencies

```bash
cd frontend
npm install
```

### Install Backend Dependencies

```bash
cd backend
npm install
```

---

## ▶️ Run Project

### Frontend

```bash
npm run dev
```

### Backend

```bash
node server.js
```

## 🌐 API Endpoint
